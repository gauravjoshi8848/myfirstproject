# JMS Bakery & Cafe Website

A responsive and modern bakery and cafe website built with **HTML, CSS, and JavaScript**.  
This project showcases a clean user interface for **JMS Bakery & Cafe**, including menu sections, special offers, location details, opening hours, and bilingual support in **English** and **Nepali**.

## Features

- Responsive design for desktop and mobile
- Bilingual interface: **English / Nepali**
- Language preference saved using **localStorage**
- Beautiful hero section and smooth navigation
- Dynamic menu rendering with categories and pricing
- Offers section for promotions
- Location section with:
  - opening hours
  - phone and email contact
  - embedded Google Map
- Clean and modern UI with custom styling
- Lazy-loaded images for better performance

## Technologies Used

- **HTML5**
- **CSS3**
- **Vanilla JavaScript**
